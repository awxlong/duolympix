CommunityRepository

CommunityRepository class abstract

Abstract base class for community data operations

Declares methods for managing collaborative features like colleague relationships, XP investments in quests, and user comments. Uses Either from the dartz package to explicitly handle success/failure cases, ensuring errors are propagated predictably.
